jSciPy

A Java library for scientific computing, inspired by SciPy.

View Javadoc View on GitHub

Overview

jSciPy fills the gap in the Java ecosystem for a comprehensive scientific computing library similar to Python's SciPy. It provides robust implementations for signal processing, optimization, and more.

Features

  • Advanced Filtering: Butterworth, Chebyshev (I & II), Elliptic, and Bessel filters. Supports zero-phase (filtfilt) and causal filtering modes.
  • 2D Processing: 2D Convolution (convolve2d), 2D FFT (fft2), and 2D IFFT (ifft2).
  • Spectral Analysis: Welch's Method (PSD), Spectrogram, standard 1D FFT/IFFT, and Real-optimized FFT (rfft/irfft).
  • Smoothing & Detection: Savitzky-Golay filters, Peak Detection (find_peaks), Median filters, Detrending.
  • Math & Analysis: RK4 ODE Solver, Linear & Cubic Spline Interpolation, Resampling.

Getting Started

Add the dependency via JitPack:

repositories {
    mavenCentral()
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.hissain:jSciPy:1.1.0'
}
